# DidactiTech

**DidactiTech** es un proyecto educativo enfocado en proporcionar recursos modernos para docentes y estudiantes. La plataforma ofrece secuencias didácticas, proyectos gamificados y acceso a tecnologías digitales para transformar la experiencia de enseñanza y aprendizaje.

## Características del Proyecto

- **Secuencias Didácticas**: Recursos diseñados para fomentar la creatividad y participación activa.
- **Proyectos Gamificados**: Propuestas innovadoras que integran juegos y tecnología para motivar el aprendizaje.
- **Acceso a Herramientas Digitales**: Descubre tecnologías para enriquecer tus clases, tanto presenciales como virtuales.

## Objetivo

El objetivo principal es crear una página web funcional con un carrito de compras que permita a los usuarios explorar y adquirir productos educativos innovadores.

## Tecnologías Utilizadas

- **HTML5**: Para la estructura de la página.
- **CSS3**: Para el diseño y los estilos.
- **JavaScript**: Para la funcionalidad interactiva, incluyendo el carrito de compras.
